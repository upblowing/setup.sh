# setup.sh

quick setup script for fresh linux installations

## what it installs
- build essentials (gcc, make)
- version control (git)
- package managers (npm, pip)
- utilities (curl, wget, neofetch, htop, vim, tmux, ffmpeg, imagemagick, zip)
- dev tools (nodejs, jq)

## usage
```bash
chmod +x setup.sh
sudo ./setup.sh
```

## supported distros
- debian/ubuntu
- fedora/rhel/centos
- arch
- MacOS
- CubeOS
